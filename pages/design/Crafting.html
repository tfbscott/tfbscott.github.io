Balancing of Digital Economies

2020

Download Excel File Here

This project is a simple economy similar to those seen in massive multiplayer online games working inside of a spreadsheet. This economy features a fleshed out crafting system, an experience system, and scaling of item power and prices.

This was my first introduction to system design. The focus was creating methods to properly balance complex interlocking systems without implementing any of the systems in a game engine. While having a game that can be tested by others is nice, sometimes you won’t have access to an engine. Knowing how to create and test systems “on paper” is an extra tool that I have used numerous times.
