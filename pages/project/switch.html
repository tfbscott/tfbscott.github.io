Urban Experiment
NX Dev Kit | 2022

Roles & Responsibilities

  • Created a render system using proprietary Nintendo Switch libraries.
  • Created a multi platform input system that ran on the Nintendo Switch and PC.

Overview

I was given the privilage of taking a course teaching the ins and outs of using a Nintendo Switch Development Kit. During the course I created multiple quick demos in custom engines and Unity that ran on provided Dev Kits. The final project was a simple 2D demolition derby called Urban Experiment. Up to 8 players control a triangle that drifts and boosts around the screen.

I used this opportunity to familiarize myself with the Switch's Input and Graphics systems. I created a quick multi-platform input system to allow for game iteration on the PC since occasionally sending a game to the Dev Kit could take minutes on the provided computers. For the final project, I followed official tutorials to create a render system using Nintendo's proprietary graphics library. This system, while quick and dirty due to the 6 week development time, supported GLSL shaders, vertex and uniform attributes, and allowed textures to be rendered via the bitmap format.
